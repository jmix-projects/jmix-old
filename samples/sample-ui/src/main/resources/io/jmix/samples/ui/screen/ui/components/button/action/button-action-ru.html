Кнопке можно назначить любое действие, содержащееся в каком-либо компоненте, реализующем интерфейс ActionsHolder, либо в самом экране.

Кнопка берет из действия значения одноименных атрибутов, если они не указаны в самой кнопке. Например, в данном случае кнопка получает заголовок из атрибута caption действия.
