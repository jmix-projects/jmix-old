PopupButton - кнопка с выпадающим списком действий.

Кнопка может содержать текст или пиктограмму (или и то и другое).

В данном примере для двух кнопок действия определены в XML. Действия третьей кнопки программно добавляются в контроллере. Эти действия получают заголовки из пакета локализованных сообщений автоматически, по ключам, соответствующим идентификаторам действий.
